# ڕێنمایی دامەزراندن و چارەسەرکردنی کێشەکان

## کێشەی WhatsApp لە iOS

### پێویستییەکان
کاتێک هەوڵی ناردنی نامە بە کڕیار دەدەیت لە ڕێگەی WhatsApp، ڕەنگە ئەم هەڵەیە ببینیت:

```
Error: Unable to open URL: whatsapp://send?phone=...
Add whatsapp to LSApplicationQueriesSchemes in your Info.plist.
```

### چارەسەر

بۆ چارەسەرکردنی ئەم کێشەیە، پێویستە `app.json` دەستکاری بکەیت و `LSApplicationQueriesSchemes` زیاد بکەیت:

```json
{
  "expo": {
    "ios": {
      "infoPlist": {
        "LSApplicationQueriesSchemes": [
          "whatsapp",
          "sms",
          "tel"
        ]
      }
    }
  }
}
```

ئەمە ڕێگە بە ئەپەکەت دەدات کە بەرنامەکانی دیکەی مۆبایل وەک WhatsApp، SMS و تەلەفۆن بکاتەوە.

## تایبەتمەندییەکانی سیستەم

### 1. بەڕێوەبردنی کڕیاران (Debtors)
- زیادکردنی کڕیاری نوێ
- دەستکاریکردنی زانیاری کڕیار
- سڕینەوەی کڕیار
- تۆمارکردنی مامەڵەکان (قەرز و پارەدان)
- وێنەی کڕیار
- تێبینی تایبەت بە کڕیار
- سنووری قەرز
- ڕێژەی سوود
- پۆلینکردن (VIP, Regular, Wholesale)

### 2. سیستەمی کارمەند
- زیادکردنی کارمەند لە لایەن بەڕێوەبەر
- دیاریکردنی مۆڵەتەکان:
  - بینینی کڕیاران
  - زیادکردنی کڕیار
  - دەستکاریکردنی کڕیار
  - سڕینەوەی کڕیار
  - زیادکردنی مامەڵە
  - دەستکاریکردنی مامەڵە
  - سڕینەوەی مامەڵە
  - بینینی ڕاپۆرتەکان
  - هەناردەکردنی داتا

### 3. سیستەمی کڕیار
- دروستکردنی هەژماری کڕیار
- بینینی قەرزی خۆی
- بینینی مامەڵەکانی خۆی
- QR کۆدی تایبەت بە کڕیار
- وەسڵی فەرمی

### 4. سکانی QR کۆد
- تەنها بەڕێوەبەر و کارمەند دەتوانن QR کۆد سکان بکەن
- گەیشتن بە زانیاری کڕیار بە خێرایی
- لە پەڕەی سەرەکی، دوگمەی QR کۆد لەبەردەستدایە

### 5. ئاگادارییەکان
- ئاگاداری خودکار بۆ نزیکبوونەوە لە سنووری قەرز
- ئاگاداری بۆ تێپەڕین لە سنووری قەرز
- ناردنی ئاگاداری بۆ کارمەندان
- ناردنی ئاگاداری بۆ کڕیاران
- ناردنی یادەوەری بە WhatsApp و SMS

### 6. ڕاپۆرت و ئامار
- کۆی گشتی قەرز
- ئامار بەپێی دوایین ٣٠ ڕۆژ
- دیاگرامی پای (Pie Chart)
- دیاگرامی هێڵ (Line Chart)
- ڕاپۆرتی تایبەت
- هەناردەکردنی داتا

### 7. ئاسایشی داتا
- PIN کۆد
- ناسینەوەی سەرەتایی (Face ID / Touch ID)
- شاردنەوەی بڕی پارە
- پاشەکەوتی خودکار
- هەناردە/هاوردە

### 8. وەسڵی فەرمی
- وەسڵێکی جوان و پڕۆفیشناڵ
- ناوی مارکێت بە گەورەیی
- ناوی کڕیار و بەرواری وەسڵ
- لیستی مامەڵەکان
- کۆی گشتی قەرز
- بەشی واژوو و مۆر
- تێبینی یاسایی
- کۆدی وەسڵ بۆ پشتڕاستکردنەوە

## چۆنیەتی بەکارهێنان

### دەست پێکردن

1. **وەک خاوەندار:**
   - چوونە ژوورەوە بە: `owner` / `owner123`
   - پەسەندکردنی داواکارییەکانی مارکێتەکان
   - درێژکردنەوەی مۆڵەتی بەشداریکردن

2. **وەک بەڕێوەبەری مارکێت:**
   - پێشکەش کردنی داواکاری مارکێت لە پەڕەی Market Request
   - چاوەڕوانی پەسەندکردن لە لایەن خاوەندار
   - چوونە ژوورەوە بە ژمارە تەلەفۆن و ووشەی نهێنی

3. **زیادکردنی کارمەند:**
   - لە پانێڵی بەڕێوەبەر، سەردانی "بەڕێوەبردنی کارمەندان" بکە
   - زانیاری کارمەند زیاد بکە
   - مۆڵەتەکان دیاری بکە

4. **زیادکردنی کڕیار:**
   - دوگمەی "+" لە پەڕەی سەرەکی بگرە
   - زانیاری کڕیار پڕ بکەرەوە
   - دروستکردنی هەژماری کڕیار (ئیختیاری)

## زانیاری دەستکاری app.json

### گرنگ: 
من ناتوانم فایلی `app.json` دەستکاری بکەم بە هۆی سنووردارکردنی سیستەمەوە.

**بۆ چارەسەرکردنی کێشەی WhatsApp، تکایە ئەم هێڵانە بە دەستی زیاد بکە:**

لە بەشی `ios.infoPlist`:
```json
"LSApplicationQueriesSchemes": [
  "whatsapp",
  "sms",
  "tel"
]
```

## تایبەتمەندییە نوێکان

### پێشنیارەکان بۆ پێشخستنی سیستەم:

1. **یەکخستن لەگەڵ خزمەتگوزارییەکانی دەرەوە**
   - یەکخستنی Cloud Backup (iCloud / Google Drive)
   - یەکخستنی پارەدان ئۆنلاین
   - API بۆ یەکخستنی لەگەڵ بانک

2. **ڕاپۆرتی پێشکەوتوو**
   - پێشبینیکردنی قەرز بە AI
   - شیکاری هەڵسوکەوتی کڕیار
   - ئاگاداری هۆشمەندانە

3. **تایبەتمەندی WhatsApp پێشکەوتوو**
   - ناردنی نامەی کۆمەڵایەتی
   - قاڵبی نامە
   - خودکارکردنی یادەوەری

4. **بەڕێوەبردنی کاڵا**
   - تۆمارکردنی کاڵا
   - کۆگای کاڵا
   - سکانی باڕکۆد

5. **تایبەتمەندییە مەترسییەکانی زیاتر**
   - ناوچەی جوگرافی کڕیار (GPS)
   - سیستەمی خاڵ و پاداشت
   - کارتی بەشداریکردن

6. **دواخستنی پارەدان**
   - خشتەی پارەدان
   - یادەوەری خودکار بۆ دواخستنەکان
   - گرێبەستی پارەدان

7. **ڕاپۆرتی باشتر**
   - چاپکردنی ڕاپۆرتی مانگانە
   - ئیمەیڵکردنی ڕاپۆرتەکان
   - شارەزایی و ڕاوێژکاری بازرگانی
